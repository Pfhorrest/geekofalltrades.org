    optional-motion

    @mixin optional-motion() { ... }

    Description

    Helper mixin that sets animations and transitions to apply only when the browser is not set to reduced motion and the data-reduced-motion attribute is not set to yes in either the html or the element, but always when the data-reduced-motion attribute is set to no in either the html or the element regardless of browser preference, without having to duplicate code.

    Parameters

    None.

    Requires

    • [variable] bp2 — Second breakpoint value

    Used by

    • [mixin] h1-large
    • [mixin] menu-item-styles
